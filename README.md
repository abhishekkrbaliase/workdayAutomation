# WorkdayAutomation
This is a chrome extensin which can fill in the Experience and education in workday. This is doesn't fill in any other forms.



## Installation
1. Edit data.js to fill in your experience
2. Open "Extension Manager" in chrome browser and enable developer mode
3. Click on "Load Unpacked"
4. Open workday and Click on relevant button to fill the details.

## Usage Warning
The plugin tries to fill in based on the labels. There can be some error in seecting appropriate dropdowns which can be filled in manually.
